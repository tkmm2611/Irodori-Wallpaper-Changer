# 彩 - Wallpaper Changer

An Android app that changes wallpapers automatically
when the screen turns on.
画面がオンになると自動的に壁紙を変更するAndroidアプリ。

## Features
- Change wallpaper on screen ON
- Select image folder
- Simple and clean UI
- [画面ON時]に壁紙を変更
- 画像フォルダを選択
- シンプルでクリーンなUI
